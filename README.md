📝 Project Overview
A comprehensive data analysis project focused on Carrefour retail sales data. It includes data cleaning, relational modeling, and interactive visualizations to track key retail KPIs, optimize product performance, and analyze customer demographics.

🎯 Business Objectives
* 📈 **Revenue Optimization:** Identify top-performing product categories, high-value customer segments, and peak sales periods.
* 👥 **Customer Insights:** Analyze purchasing patterns across different demographics, payment methods, and branch locations.
* ⚙️ **Operational Efficiency:** Evaluate sales trends over time to assist in predictive demand planning and inventory allocation.

 🛠️ Tech Stack & Tools
* 🧼 **Power Query / SQL (T-SQL):** Data Cleansing & Transformation
* 🏗️ **Power BI:** Data Modeling (Star Schema / Relational Modeling)
* 🧮 **DAX / Advanced Excel:** Analytical Calculations & Measures
* 📊 **Power BI Desktop:** Interactive Dashboards & Visualizations

📊 Key Insights & Features
* 💰 **Executive Summary:** Tracking total revenue, total transactions, gross profit margins, and average order value (AOV).
* 📦 **Product Performance:** Breakdown of sales by product line, identifying best-sellers and slow-moving items.
* ⏰ **Temporal Sales Trends:** Hourly, daily, and monthly analysis highlighting peak shopping hours and seasonal trends.
* 🏢 **Demographic & Branch Analysis:** Comparative performance across different city branches and customer types (Member vs. Normal).

📂 Project Files
* 📊 **Carfour.pbix:** The main interactive Power BI dashboard containing the visual reports and data model.
* 📄 **Carrefour Presentation.pdf:** A structured presentation summarizing the key insights and recommendations derived from the data.
